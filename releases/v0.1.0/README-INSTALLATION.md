# YouTubeMonitor - M1 Mac 版本

## 📦 安装说明

### 系统要求
- macOS 11.0 (Big Sur) 或更高版本
- Apple Silicon (M1/M2/M3) Mac

### 安装步骤

1. **下载安装包**
   - 下载 `YouTubeMonitor_0.1.0_aarch64.dmg`

2. **安装应用**
   - 双击 `.dmg` 文件
   - 将 `YouTubeMonitor.app` 拖拽到 `Applications` 文件夹

3. **首次运行**
   - 打开 `Applications` 文件夹
   - 右键点击 `YouTubeMonitor.app`
   - 选择"打开"（首次运行需要此步骤）
   - 在弹出的对话框中点击"打开"

### 功能特点

✅ **YouTube 频道监控**
- 监控多个 YouTube 频道
- 自动获取最新视频
- 支持分组管理

✅ **数据分析**
- 视频播放量分析
- 频道增长趋势
- 数据可视化图表

✅ **视频下载**
- 一键下载视频
- 支持批量下载
- 自动分类存储

✅ **智能提醒**
- 新视频通知
- 自定义刷新频率
- 收藏功能

### 首次使用

1. **软件激活**
   - 打开应用后，进入"设置 → 软件激活"
   - 复制您的机器码
   - 联系管理员获取激活码
   - 输入激活码完成激活

2. **添加频道**
   - 点击右上角"添加频道"按钮
   - 粘贴 YouTube 频道 URL
   - 支持批量添加（每行一个 URL）

3. **配置设置**
   - 设置 → 常规设置
   - 配置视频下载路径
   - 设置代理（如需要）
   - 配置 Cookie（解决 "Sign in" 报错）

### 常见问题

**Q: 提示"无法打开应用"怎么办？**
A: 右键点击应用，选择"打开"，然后在弹出的对话框中点击"打开"。

**Q: 如何获取激活码？**
A: 联系管理员，提供您的机器码即可获取激活码。

**Q: 视频下载失败怎么办？**
A: 
1. 检查网络连接
2. 配置代理（如在中国大陆）
3. 配置 Cookie（参考设置页面说明）

**Q: 数据存储在哪里？**
A: 应用数据存储在 `~/Library/Application Support/YouTubeMonitor/`

### 卸载

1. 将 `YouTubeMonitor.app` 从 Applications 文件夹移到废纸篓
2. （可选）删除应用数据：
   ```bash
   rm -rf ~/Library/Application\ Support/YouTubeMonitor
   ```

### 更新

下载新版本的 `.dmg` 文件，覆盖安装即可。应用数据会自动保留。
