# 激活系统使用指南 (Activation System Guide)

您已经成功实施了基于机器码绑定的激活系统。

## 如何运作
1.  **用户端**: 用户打开 APP -> 设置 -> 软件激活。
2.  **获取机器码**: 用户会看到“本机机器码 (Machine ID)”，例如 `59258273-0948-...`。
3.  **发送给您**: 用户将此代码复制发给您。
4.  **生成激活码**: 您在本地运行生成脚本得到激活码。
5.  **激活**: 用户输入激活码，点击保存。系统验证通过后解锁所有功能。

## 管理员操作 (生成激活码)
在项目根目录下，使用 Node.js 运行 `admin-keygen.js` 脚本。

### 命令格式
```bash
node admin-keygen.js <用户的机器码> [有效期天数]
```
默认有效期为 1 年 (365 天)。

### 示例
```bash
# 默认激活 (1年)
node admin-keygen.js 59258273-0948-4309-8733-1234567890

# 激活 30 天
node admin-keygen.js 59258273-0948-4309-8733-1234567890 30
```

### 输出
```text
========================================
Machine ID:       59258273-0948-4309-8733-1234567890
Activation Code:  6ED35C-7E2EDC-123ABC-456DEF
========================================
```
将 `6ED35C-7E2EDC-123ABC-456DEF` 发给用户即可。

> [!CAUTION]
> **密钥安全**: 生成逻辑依赖 `src/lib/activation.ts` 和 `admin-keygen.js` 中的 `SECRET_KEY`。请务必保管好此密钥，不要泄露。如果您更改了密钥，旧的激活码将全部失效。
